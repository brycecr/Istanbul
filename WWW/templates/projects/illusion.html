Decomposition

This problem has a logical group of commands that draws a square that should be decomposed into its own method. The challenge is, that the method needs extra information, the row and column where it should be drawn. This problem can be addressed by using parameter passing. In your program, implement and use a method with the following prototype:

private drawSquare(int row, int col) {
   //code for drawing a square at the position row, col
}
